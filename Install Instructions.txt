1. How to make ipywidgets work in jupyter lab?
Answer.
Step 1. Download and install node.js
Step 2 - Open "Anaconda Prompt" as an administrator
Step 3 - Execute the below command:
jupyter labextension install @jupyter-widgets/jupyterlab-manager

2. How to install voila?
Answer. 
Step 1 - Open "Anaconda Prompt" as an administrator
Step 2 - Execute the below command:
conda install -c conda-forge voila

3. How to install JupyterLab preview extension?
Answer
Step 1 - Open "Anaconda Prompt" as an administrator
Step 2 - Execute the below command:
jupyter labextension install @jupyter-voila/jupyterlab-preview
